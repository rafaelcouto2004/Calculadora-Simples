# 🧮 Calculadora - Rafael Rocha

Uma calculadora moderna, responsiva e funcional desenvolvida com foco em design limpo e experiência do usuário.

## ✨ Funcionalidades
- **Modo Escuro/Claro:** Alternância de tema com persistência.
- **Histórico:** Registra os últimos cálculos realizados.
- **Segurança:** Processamento de operações sem o uso de `eval()`.
- **Suporte a Teclado:** Totalmente funcional via teclado físico do computador.
- **Responsividade:** Interface adaptada para desktop e dispositivos móveis.

## 🛠️ Tecnologias Utilizadas
- **HTML5** (Estrutura semântica)
- **CSS3** (Grid Layout, Flexbox e Variáveis de ambiente)
- **JavaScript** (Manipulação de DOM e LocalStorage)
